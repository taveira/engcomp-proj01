<p>O modelo de classes é um modelo lógico do sistema de software em construção. As classes geralmente têm uma relação direta com o código-fonte ou outros artefatos de software que podem ser agrupados em componentes executáveis.<br /></p><p><br /></p><p>O pacote do sistema contém as classes e artefatos que estão sendo construídas ou projetadas como parte do modelo atual. O pacote Frameworks geralmente contém classes e componentes que foram projetados e construídos anteriormente e estão sendo reutilizados como parte do projeto atual.<br /></p>
